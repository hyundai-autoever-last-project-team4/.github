# 📌 프로젝트 소개
> **차량 부품의 입고부터 출고까지 전체 프로세스를 관리하고 운영할 수 있는 시스템**

- 대규모 창고 시스템을 **효율적으로 관리**하고 **실시간 모니터링**
- 사용자 역할별 **입고/출고/적치 프로세스 지원**
- 창고 현황을 **실시간 알림**을 통해 파악 가능
